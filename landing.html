Summary: Latitude vs. X

This project involved using API calls to collect weather data on a list of random cities, and then using this data to visualize and make observations on the correlation between a citie's latitude and its weather patterns.

We first used a Python library citipy to generate a list of random cities based on random coordinates. Then we gather the datasets and compare latitude against 4 factors: Maximum Temperature, Cloudiness, Maximum Wind Speed, and Humidity. Finally, we plot using Matplotlib, and make observations on any correlations.
